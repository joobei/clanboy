# clanboy

Clanboy is a clan management website initially designed for Hell Let Loose, can be adapted to any clan.

## Project setup
```
npm ci
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
